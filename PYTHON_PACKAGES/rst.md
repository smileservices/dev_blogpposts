# RST syntax


- Headings
Each doc section must have a declared title 

    Title
    =====

    Section
    -------

    Subsection
    ~~~~~~~~~~


- Code samples

    You can use ``backticks`` for showing ``highlighted`` code.


- Hyperlinks

    `A cool website`_
    .. _A cool website: http://sphinx-doc.org


- Code Example Syntax

    A cool bit of code::
    Some cool Code
    .. code-block:: rst
    A bit of **rst** which should be *highlighted* properly.


- References

    .. _reference-name:

    Cool section
    ------------
    :ref:`reference-name`
